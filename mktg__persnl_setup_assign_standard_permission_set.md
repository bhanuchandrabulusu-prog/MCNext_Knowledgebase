# Assign a Standard Permission Set

**Source:** https://help.salesforce.com/s/articleView?id=mktg.persnl_setup_assign_standard_permission_set.htm&language=en_US&type=5

---

SALESFORCE PERSONALIZATION
Assign a Standard Permission Set

Select a predefined set of permissions to assign Personalization Admin or Personalization user roles to users.

REQUIRED EDITIONS
USER PERMISSIONS NEEDED
To create permission sets	Manage Profiles and Permission Sets
To assign permission sets	

Assign Permission Sets

View Setup and Configuration

In Setup, search for and select Users.
Select a user.
In the Permission Set Assignments section, click Edit Assignments.
Select the Personalization Admin or Personalization User permission set.
Click Add and then save your work.
