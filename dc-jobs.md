# Data 360 Data- and Development-Related Jobs | Get Started with Data 360 Development | Data 360 Developer Guide

**Source:** https://developer.salesforce.com/docs/data/data-cloud-dev/guide/dc-jobs.html

---

Planning your data strategy and importing and mapping data are part of the prerequisite tasks that you perform before you can create insights and build custom apps. Each of these phases require a different job role.

Data administrator—A data administrator in your organization analyzes the data to import to Data 360 and defines the data strategy.
System integrator—You can hire a system integrator from an external software consulting company to help with the setup of Data 360. The system integrator interfaces with the data administrator to analyze and map the data.
Salesforce partner—Salesforce partners build custom apps for Data 360 and Salesforce. You can purchase an app from AppExchange to help with automating processes.
Customer developer—A customer developer is a developer in your organization who builds custom apps.
