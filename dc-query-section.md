# Get Started With Data 360 SQL | Data 360 Query Guide

**Source:** https://developer.salesforce.com/docs/data/data-cloud-query-guide/guide/dc-query-section.html

---

This section provides examples of commonly used SQL queries in Data 360 and helps you learn about some of the queries you can write.

Let’s consider this scenario: As an administrator, you create a data stream in Data 360 by using the Salesforce CRM data bundle. Now, you want to get the useful information and insights out of the data in Data 360 by running SQL queries. This section walks you through various scenarios of fetching data and presents example SQL queries. To get started, let’s write a simple query.

To learn more about data bundles, see Model Data in Data 360.

User Permissions Sets Needed	
To query Data 360 Objects	Data Cloud Activation Manager

To query a Data 360 object, you must have dataspace access to that object. For more information about data space aware objects, see Data Space Aware Objects and Features.
