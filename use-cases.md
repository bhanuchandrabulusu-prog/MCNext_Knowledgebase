# Use Case Examples | Data 360 Connect REST API

**Source:** https://developer.salesforce.com/docs/data/connectapi/guide/use-cases.html

---

Use the Data Model Object (DMO) API to programmatically define, update, and manage the mapping of source data to your unified data model.

Data Streams: Bulk Disable Salesforce CRM Fields

Use the Data Streams API to identify and bulk disable inaccessible or deprecated fields in Salesforce CRM connector data streams to maintain data stream health and minimize deploy cycles.

Query: Retrieve Campaign Engagement Records

Use the Query API to perform a comprehensive audit and retrieve large volumes of data efficiently by managing the asynchronous pattern, checking job status via polling, and handling results using pagination.
