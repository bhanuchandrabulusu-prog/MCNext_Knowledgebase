With the Summer ’25 release of **Marketing Cloud Engagement**, a brand-new feature called the **“Data Extension Usage Dashboard”** has been introduced. This dashboard offers a visual and intuitive way to understand how data storage is being used within your account.

📊 **How to Access It**  
 You can access the dashboard by navigating to:  
 **Setup > Data Management > Data Extension Storage**

This new dashboard is a helpful tool for visualizing and managing your growing data, making it easier to optimize storage and make cleanup decisions.

## Final Thoughts

The new **Data Extension Usage Dashboard** is incredibly useful for instantly grasping your account’s storage situation at a glance. However, for more **detailed analysis**, downloading the **CSV file** remains the most effective method.

🔽 **How to Download the CSV File**  
You can download it by clicking the **“Download Storage Details”** link at the bottom-left corner of the **Setup** screen.

**Note**: If the link is **grayed out**, it’s likely because you don’t have access to all Business Units in the account (including child BUs). In that case, please ask your admin to download it for you.

Another great benefit: the CSV file also includes the **folder paths** for each Data Extension — perfect for when you find yourself wondering, “Which folder did I save that in…?”

I was secretly hoping that a **folder path search feature** would be included in this new dashboard by default… but unfortunately, that hasn’t been implemented yet. 😅
