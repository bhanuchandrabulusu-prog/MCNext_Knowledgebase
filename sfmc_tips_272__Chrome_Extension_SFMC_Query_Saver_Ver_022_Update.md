## **Update Details**

Version 0.2.2 (2026/03/18)

![]()

Now, the details of this update are as follows.

- **1. CSV Import**  
  You can now import queries if you have exported them from another Chrome environment.
- **2. able Link**  
  You can generate a shareable link valid for 24 hours and share it with team members.
- **3. Smart Grouping**  
  Saved SQL queries are organized and grouped by date, making them easier to find.

> *If this update is not yet visible, please access chrome://extensions and click “Update.”*

## **1. CSV Import**

You can now import queries exported from another Chrome environment.

- **“CSV Export” and “CSV Import” buttons have been added to the top right of the screen.**
- **Easy migration and backup of queries.**

## **2. able Link**

You can now generate a shareable link valid for 24 hours for each query.

- **One-click copy of the link.**
- **Easy sharing with team members.**

When you open the shared URL, you can check the query content in the browser.

## **3. Smart Grouping**

Saved SQL queries are now automatically organized by date.

- You can see at a glance when the query was used.
- Significantly improved searchability of past queries.

It has evolved from the previous simple UI into a more practical and user-friendly interface.
