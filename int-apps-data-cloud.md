# Data 360 Integrated Apps | Query Data in Data 360 | Data 360 Query Guide

**Source:** https://developer.salesforce.com/docs/data/data-cloud-query-guide/guide/int-apps-data-cloud.html

---

To query Data 360 data with a UI, use integrated apps. Create and run SQL statements with the support of one of these apps.

Query Editor. You can use Query Editor in Data 360 to write and execute SQL queries. See Query Editor in Salesforce Help.
Tableau. Tableau is a visual analytics platform that transforms the way data solves problems. See What is Tableau?.
DBeaver. DBeaver is a universal database management tool that you can use to query your Data 360 data. See Connecting to Salesforce Data 360 in DBeaver.
