SALESFORCE PERSONALIZATION
Troubleshooting Experiment Analytics

Use the information in this section to supplement any experiment analytics errors.

ISSUE	RESOLUTION
No data or metrics appear for the experiment	Make sure to allow ‌enough time for an experiment to provide results. Experiments require up to 24 hours from the time they start to process data and display analytics. If this issue persists, check the experiment setup. Make sure that any required data lake objects for experimentation are added and properly mapped for the data space.
