# Update the Data 360 Connector Permissions

SALESFORCE PERSONALIZATION
Update the Data 360 Connector Permissions

Configure the Data 360 permission set with the appropriate Salesforce Personalization access and permissions.

REQUIRED EDITIONS
USER PERMISSIONS NEEDED
To update permission sets:	Manage Profiles and Permission Sets
In Setup, search for and select Permission Sets.
Select Data Cloud Salesforce Connector.
Grant access to the Personalization platform.
Under System, select System Permissions.
Enable Access Personalization Platform.
Save your changes.
On the Permission Set Overview page, under Apps, click Object Settings.
Grant Read and View All Records permissions for these Data 360 objects.
Data Graph Definitions
Data Model Objects
Data Spaces
Grant Read and View All Records permissions for these Personalization objects.
