SALESFORCE PERSONALIZATION
What the Use Case Setup Deploys

You can currently select from two different use cases to deploy – Maximize Revenue and Maximize Clicks. Each use case creates and deploys different assets.

USE CASE	DEPLOYED ASSETS	OBJECT NAME
Maximize Revenue Products	Profile Data Graph	Real-Time Data Graph
 	Item Data Graph	Product Data Graph
 	Content Schema	Product Recommendations Schema
 	Recommender	Maximize Revenue Product Recommender
 	Personalization Point	Product Recommendations
 	Calculated Insight	Top Sellers
Maximize Clicks Articles	Profile Data Graph	Real-Time Data Graph
 	Item Data Graph	Article Data Graph
 	Content Schema	Product Recommendations Schema
 	Recommender	Maximize Clicks Article Recommender
 	Personalization Point	Article Recommendations
