SALESFORCE PERSONALIZATION
Create a Segment for Salesforce Personalization

You can create standard or real-time segments to use for decision targeting.

To create a standard segment, see Create a Segment.

To create real-time segments for Salesforce Personalization, use these instructions.

In Data 360, go to the Segments tab and click New.
Select Create with Visual Builder, and select Real-Time Segment.
Click Next.
Enter the segment details.
Select the data space that the source data graph for this new segment is assigned to.
You can’t assign a segment to a different data space than its source data object.
For Segment On, select the real-time data graph that you’re basing the segment on.
You can select attributes to filter segment membership from the fields in the data graph.
Give your segment a unique, easily identifiable name.
Click Next.
Set the time period over which the real-time segment can be built, and click Next.
Select the data graph fields on which segment membership is filtered.
Save your work.
