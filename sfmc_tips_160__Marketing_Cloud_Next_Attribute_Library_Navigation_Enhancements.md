### **Key Features**

- The new attribute library can now be navigated using a **folder-like hierarchical structure**. You can access attributes within a DMO one level at a time.
- While navigating, the **breadcrumb trail** in the header displays the current path, making it easy to return to the original location.
- The new design also **distinguishes between DMO, Calculated Insights, and Attributes**.
- When adding a container, you can choose to aggregate either at the parent object level or the current object level. This enhancement reduces payload size, improves load time, and enhances overall performance.

### **How to Enable**

In **Data Cloud** settings, search for **Feature Manager**, and enable the feature **“Navigate the Attribute Library More Intuitively”.**

## Conclusion

This update introduces a more intuitive hierarchical structure with the target DMO at the top, smoother navigation back to the original DMO via breadcrumbs, and clearer separation between DMO, Calculated Insights, and Attributes — bringing several small but meaningful improvements.

Although it’s a minor update, it enhances usability and makes the experience more user-friendly, so I recommend giving it a try.
