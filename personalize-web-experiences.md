# Personalize Web Experiences

**Source:** https://developer.salesforce.com/docs/marketing/einstein-personalization/guide/personalize-web-experiences.html

---

Salesforce Interactions SDK enables you to capture customer interactions on your website and send that data to Data Cloud. In Data Cloud, you can use this data to build behavior profiles for website visitors, segment audiences, or integrate that data with other Salesforce products.

You can also use the Interactions SDK to send customer interactions to Personalization as requests through the Decisioning API. Personalization processes these requests and returns personalized content that you can render and display on specific areas of your website.

The SDK uses the SalesforceInteractions namespace, which encapsulates its core components. Personalization extends this namespace with the Personalization object, providing methods and functions to interact with its features.
