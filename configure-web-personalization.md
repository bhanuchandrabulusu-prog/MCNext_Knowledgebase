# Configure the Personalization Module | Personalize Web Experiences

**Source:** https://developer.salesforce.com/docs/marketing/einstein-personalization/guide/configure-web-personalization.html

---

Salesforce Interactions SDK includes the Personalization module that enables you to define and customize how personalized data received from Personalization is rendered on your website. This module includes objects using which you can publish personalized experience configurations that business users generate using Web Personalization Manager. Additionally, you can also configure webpage flicker defense and specify additional destinations to which you can route engagement data.
