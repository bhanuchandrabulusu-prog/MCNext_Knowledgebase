## Winter ’26 — New Marketing Performance Features

### ① Insights Tab

The Marketing Performance dashboard includes two tabs: **Insights** and **Sends**. In Winter ’26, two new components have been added to the **Insights** tab, enabling even more precise, data-driven decision-making.

- **New Chart:** Easily identify user segments with the highest performance.
- **Line Graph:** Track multi-channel activity trends over time.

### ② **Deliverability** Tab

The **Deliverability** tab has also been updated. With the redesigned interface, marketers can now conduct a more detailed analysis of their **email campaign performance**.

### Domain Filter

A new **Domain Filter** feature allows you to visualize **delivery failure reasons** by recipient domain — such as Gmail or Hotmail — helping diagnose deliverability issues more efficiently.

### Message Type Filter

Another new capability, the **Message Type Filter**, lets you separate and analyze the performance of **promotional emails** versus **transactional emails**.

## When Winter ’26 Features Don’t Appear

In some environments, the new **Marketing Performance** features from **Winter ’26** may not be immediately visible.

If this happens, **reinstalling the feature** may resolve the issue. Note that performing a reinstallation **will not affect your existing historical data**.

### Re-installation Steps

**1. Remove the installation from the settings page**  
In the setup search bar, type **“Marketing Performance”** and select the Marketing Performance tab. Then, click the **Delete** button in the upper-right corner of the screen.

**2. Re-install the feature**  
Once the deletion is complete, the **Install** button will become active again. Click it to reinstall the feature.

## **Conclusion**

The **Marketing Performance** feature can also be accessed **directly from campaign records**. When used from a campaign record, the dashboard automatically filters results to display data for that specific campaign.

Please note the following points when using this feature:

- The dashboard’s layout and content **cannot be customized**.
- Using this feature **consumes Data Cloud credits**.
