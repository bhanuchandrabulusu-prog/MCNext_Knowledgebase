Salesforce has officially released **two comprehensive Quick Start Guides** to help users smoothly begin their journey with **Marketing Cloud Next** (Marketing Cloud Growth & Advanced Edition).

※ In August 2025, the **Quick Start Guide for SMS** was published.  
 The link is [here](https://help.salesforce.com/s/articleView?id=mktg.mktg_quick_start_parent_sms.htm&language=en_US&type=5).

These guides provide clear instructions on tasks ranging from setting up the data environment, configuring the email channel, and installing reporting tools to importing contact and consent data and sending emails.

Previously, users needed to refer to multiple help documents to achieve these configurations. However, with these Quick Start Guides, you can efficiently and systematically start leveraging Marketing Cloud Growth & Advanced Edition. The guides include abundant screenshots and related resources to support an organized and streamlined process.

## Quick Start Guide 1: Set Up Email Sending

[## Help And Training Community

### Edit description

help.salesforce.com](https://help.salesforce.com/s/articleView?language=en_US&id=mktg.mktg_admin_email_quick_start.htm&type=5&source=post_page-----3bf835cc889a---------------------------------------)

This guide focuses on the initial setup required to start sending emails through Marketing Cloud. To keep the implementation simple and fast, advanced configurations such as Einstein and Data Graph settings are excluded, with the emphasis placed on the most fundamental setup.

**What’s included in this guide:**

- Identify Admins and Assign Permissions
- Enable Marketing Cloud
- Install Data Kits
- Configure Identity Resolution
- Set Up Email
- Set Up Reporting

**Note:**  
For domain authentication for sending emails, collaboration with IT teams or external vendors is usually required.

## Quick Start Guide 2: Send an Email Campaign

help.salesforce.com](https://help.salesforce.com/s/articleView?language=en_US&id=mktg.mktg_campaigns_message_quick_start.htm&type=5&source=post_page-----3bf835cc889a---------------------------------------)

This guide walks you through the steps to import leads and contacts, set up consent data, and send promotional emails. Designed with first-time users in mind, it simplifies the process into straightforward, basic steps.

![]()

**What’s included in this guide:**

- Import Leads and Contacts
- Import Consent Data
- Create an Email Campaign
- Add a Segment to a Campaign
- Customize the Default Email
- Preview and Test Your Email
- Publish Your Email
- Send or Schedule Your Email
- Review Email Performance

These Quick Start Guides are excellent resources for those new to Marketing Cloud Growth & Advanced Edition, offering a straightforward path to efficiently complete the initial setup. By following these simplified steps, you can start utilizing the platform effectively from day one.

I hope these guides assist you in your journey with Marketing Cloud Growth & Advanced Edition. Give them a try and streamline your setup process!
