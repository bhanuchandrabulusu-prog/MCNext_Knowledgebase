Standard data bundles contain metadata provided by Salesforce that define the structure and relationships for specific business domains or platform features. They provide a standardized way to model and organize data for various industries and use cases. Standard data bundles only include standard mappings and standard DMOs.

View the list of available data bundles below:

CMS Base
Field Service
Financial Aid
Service

To use these data bundles:

Review the documentation for the specific data bundle you want to use
Understand the entity structure and relationships
Configure your data sources to match the defined schema
Implement the data model in your Data 360 environment
