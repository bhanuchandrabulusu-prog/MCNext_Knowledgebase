In Marketing Cloud Next Growth & Advanced Edition, by installing “**Marketing Performance**” **powered by Tableau Next**, the following two analytics tools became available:

- **① “Marketing Performance” dashboard**
- **② “Campaign Performance” dashboard**

*\*In the Summer ’26 new feature release, the official name was changed from Marketing Performance to* ***Marketing Performance Intelligence****.*

## 1. Marketing Performance Dashboard

This dashboard can be accessed from the “**Marketing Performance**” tab. It displays performance data such as the number of opens and clicks, segmented by campaign, channel, flow, and segment.

Additionally, it visualizes detailed metrics, including click-through rates, delivery rates, bounce rates, and unsubscribe rates.

## 2. Campaign Performance Dashboard

This dashboard allows you to dive deeper into the performance of specific campaigns. Accessed via the “**Insights**” menu on the left side of a campaign page, it enables you to filter by date range, flow name, segment name, and channel (Email, SMS, WhatsApp) to visualize results. This makes it easy to analyze detailed metrics within a specified time frame.

In addition to visualizing metrics, the dashboard also displays a list of open and click activities within the campaign. It includes features to capture and copy screenshots or download them as images.

## Prerequisites

**1. Installation of the “Marketing Data Kit” (Required)**  
The “Marketing Data Kit” must be installed beforehand.

**2. Web Tracking Data Integration (Optional)**  
If you want to include web tracking data in your reports, the web tracking functionality must be configured in advance.

**3. Permission Set**  
To access these features, users need the following permission set:

- **Tableau Next Included App Business User**

![]()

**Note:** Only users with this permission set can access the “Marketing Performance” tab and dashboards.

## Points to Note

- Dashboards cannot be customized in terms of content or layout.
- Using these features will consume Data Cloud credits. If these analytics tools are unnecessary for your operations, consider refraining from installation.

## Installation Guide

1. Navigate to **Setup**
2. Select **Marketing Cloud > Marketing Features > Marketing Performance**
3. Execute the installation

*Since Summer ’26, you can now see the installation progress and track how far the installation process has advanced.*
