SALESFORCE PERSONALIZATION
Configure a Real-Time Insight for Individual Interactions (Example)

The Individual Interactions Count real-time insight provides the number of events associated with a unified profile. You use the Visual Builder to create a real-time insight.

In Data 360, use the Visual Builder to create a real-time insight and select Count for the measure.
Search for the Shopping Cart Engagement object, and select the Shopping Cart Engagement Id field.
Click Next.
Name the measure Engagement Count, and click Apply.
To save the aggregate node, click Apply.
Name the insight Individual Interactions Count.
The Insight API Name field is auto-filled.
Click Save and Enable.
