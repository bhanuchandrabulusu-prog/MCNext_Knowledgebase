If you’re currently using Marketing Cloud Engagement+, as of **July 28, 2025**, you can now view the message usage data sent via Marketing Cloud Engagement+ through the **Digital Wallet**.

This makes it easy to visually track, via a dashboard, how many message credits (which replace the former SuperMessages) have been consumed, broken down by channels such as email and SMS.

This functionality appears to be the feature announced at **Dreamforce 2024** and has now been officially released. However, based on the current implementation, it seems that **real-time SuperMessage usage for the standard Marketing Cloud Engagement** still cannot be tracked via this new dashboard.

*From Dreamforce 2024*

## What is Marketing Cloud Engagement+?

**Marketing Cloud Engagement+** is an enhanced version of the traditional Marketing Cloud Engagement. It includes access to **Marketing Cloud Next**, Salesforce’s next-generation marketing platform.

**Marketing Cloud Engagement+** GA Date: **July 17, 2025**

The “plus” in Engagement+ signifies its integration capabilities with Marketing Cloud Next and the support of AI agents, which provide:

- Creation of highly personalized segments
- AI-powered campaign and content generation
- Suggestions for the most relevant **next best actions** for each customer

All of these enhancements significantly boost marketing productivity and performance.

### How to Access Engagement+

- **Existing MCE Users**: You can switch to the Engagement+ SKU at the time of contract renewal. It is also possible to switch during the contract period.
- **New MCE Users**: Please select the Engagement+ SKU when signing a new contract.

*Note: Implementing Data Cloud is mandatory.*

### Migrating to Engagement+

To use Engagement+, you’ll need to update your contract to the new SKU. However, this change can be made not only at the time of contract renewal but also during the contract period.

## Message Credits and How to Track Them

In **Marketing Cloud Engagement+**, every time you send a message, **message credits** are consumed. You can think of this message credit system as the replacement for the traditional **SuperMessage** used in the original Marketing Cloud Engagement.

Here are the key differences between message credits and SuperMessages:

- **Message credits are consumed** not only when messages are sent via Marketing Cloud Engagement, but also from other Salesforce products like **Marketing Cloud Next** or **Service Cloud**.
- **Usage data can be tracked in real time** through the **Digital Wallet**.

\*In traditional Marketing Cloud Engagement, you needed to contact your **Account Executive** to generate a manual, dedicated report to find out your SuperMessage usage.

### What You Can Do with the Digital Wallet

The **Digital Wallet** provides access to the following information:

- Total number of messages sent (within a selected time period)
- Remaining message credits (broken down by channel)
- Message volume by **Business Unit** and **recipient country**

This allows you to understand overall message usage trends within your organization and helps inform decisions about whether your current subscription plan needs to be adjusted.

### Available Editions and How to Check Your Access

This **Digital Wallet** feature is available in the following editions:

- Marketing Cloud Engagement+ Pro
- Marketing Cloud Engagement+ Corporate
- Marketing Cloud Engagement+ Enterprise

If you’re currently using **Marketing Cloud Engagement** and your contract renewal period is approaching, it’s a good idea to consult your **Account Executive** about switching to Marketing Cloud Engagement+.
