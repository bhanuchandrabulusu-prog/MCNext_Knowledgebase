Description: An abbreviation of the body or description suitable for display in a list. For an email, the snippet can be the first part of the body of the message.
Field API Name: std__SubjectText__c
Data Type: TEXT
Description: The name of the activity record. For example, the SubjectText for a voice call is VC-00000001. Maximum size is 255 characters.
