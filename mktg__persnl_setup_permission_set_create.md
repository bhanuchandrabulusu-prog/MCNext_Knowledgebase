SALESFORCE PERSONALIZATION
Customize Permission Sets

While the standard permission sets cover most use cases, you can clone a permission set to create a custom version with more granular control.

REQUIRED EDITIONS
USER PERMISSIONS NEEDED
To manage permission sets:	Manage Profiles and Permission Sets
From Setup, go to Permission Sets.
Click Clone next to either the Personalization Admin or Personalization User set.
Give the new permission set a unique name and save it.
Customize the system permissions and object settings as needed.
